## 1. Introduction

ProctoFlex AI est une application de surveillance sécurisée pour les examens pratiques en ligne. Cette application garantit l'intégrité académique tout en vous permettant d'utiliser les logiciels nécessaires à votre examen.

### 1.1 Fonctionnalités Principales

- **Authentification sécurisée** par reconnaissance faciale
- **Verrouillage intelligent** des applications non autorisées
- **Surveillance en temps réel** (webcam, microphone, écran)
- **Interface intuitive** pour un passage d'examen fluide
- **Sécurité RGPD** avec protection de vos données

### 1.2 Compatibilité

- **Windows 10/11** (64 bits)
- **macOS 12+** (Intel et Apple Silicon)
- **Webcam** : Résolution minimale 720p
- **Microphone** : Qualité standard
- **Réseau** : Connexion Internet stable

## 2. Installation

### 2.1 Téléchargement

1. **Accédez au portail étudiant**
   - Connectez-vous à votre espace étudiant
   - Téléchargez l'application ProctoFlex AI

2. **Vérifiez les prérequis**
   - Système d'exploitation compatible
   - Espace disque : 2 GB minimum
   - RAM : 4 GB minimum

### 2.2 Installation Windows

1. **Exécutez l'installateur**
   ```
   ProctoFlex-AI-Setup.exe
   ```

2. **Suivez l'assistant d'installation**
   - Acceptez les conditions d'utilisation
   - Choisissez le dossier d'installation
   - Autorisez l'installation des composants système

3. **Redémarrage recommandé**
   - Redémarrez votre ordinateur après l'installation
   - Cela garantit le bon fonctionnement du verrouillage

### 2.3 Installation macOS

1. **Ouvrez le fichier .dmg**
   ```
   ProctoFlex-AI.dmg
   ```

2. **Glissez l'application dans Applications**
   - Suivez les instructions à l'écran
   - Autorisez l'installation depuis des développeurs non identifiés

3. **Premier lancement**
   - Clic droit sur l'application → Ouvrir
   - Autorisez l'accès aux permissions système

## 3. Première Utilisation

### 3.1 Configuration Initiale

1. **Lancement de l'application**
   - Double-cliquez sur l'icône ProctoFlex AI
   - L'application se lance automatiquement

2. **Test de compatibilité**
   - **Webcam** : Autorisez l'accès à la caméra
   - **Microphone** : Testez votre micro
   - **Réseau** : Vérification de la connexion

3. **Configuration de la reconnaissance faciale**
   - Prenez une photo de votre visage
   - Suivez les instructions pour l'éclairage
   - Confirmez la reconnaissance

### 3.2 Authentification

1. **Connexion à votre compte**
   - Saisissez vos identifiants étudiants
   - Email : votre.email@esprim.tn
   - Mot de passe : votre mot de passe

2. **Vérification d'identité**
   - Prenez une photo de votre pièce d'identité
   - Assurez-vous que la photo soit claire et complète
   - Confirmez l'upload

3. **Validation automatique**
   - L'IA compare votre visage avec la pièce d'identité
   - Validation en quelques secondes
   - Accès à votre tableau de bord

## 4. Passage d'un Examen

### 4.1 Préparation

1. **Vérification de l'environnement**
   - **Bureau** : Nettoyez votre espace de travail
   - **Éclairage** : Assurez un éclairage uniforme
   - **Bruit** : Évitez les sources de bruit
   - **Connexion** : Vérifiez votre connexion Internet

2. **Matériel requis**
   - **Ordinateur** : Fonctionnel et chargé
   - **Webcam** : Positionnée correctement
   - **Microphone** : Testé et fonctionnel
   - **Pièce d'identité** : À portée de main

### 4.2 Démarrage de l'Examen

1. **Accès à l'examen**
   - Connectez-vous à ProctoFlex AI
   - Sélectionnez l'examen dans votre liste
   - Cliquez sur "Commencer l'examen"

2. **Vérification d'identité**
   - Reconnaissance faciale automatique
   - Vérification de la pièce d'identité
   - Validation par l'IA

3. **Configuration de la surveillance**
   - **Webcam** : Positionnement automatique
   - **Microphone** : Test de niveau sonore
   - **Écran** : Autorisation de capture

### 4.3 Pendant l'Examen

1. **Interface d'examen**
   ```
   ┌─────────────────────────────────────┐
   │ [Temps restant: 02:45:30] [Soumettre] │
   ├─────────────────────────────────────┤
   │                                     │
   │        Zone de travail              │
   │                                     │
   └─────────────────────────────────────┘
   ```

2. **Applications autorisées**
   - Seules les applications configurées sont accessibles
   - Les autres applications sont automatiquement fermées
   - Notifications en cas de tentative d'ouverture

3. **Surveillance active**
   - **Indicateur vert** : Surveillance normale
   - **Indicateur orange** : Comportement suspect détecté
   - **Indicateur rouge** : Incident détecté

### 4.4 Fin d'Examen

1. **Soumission**
   - Cliquez sur "Soumettre l'examen"
   - Confirmez la soumission
   - Attendez la confirmation

2. **Transmission des données**
   - Upload automatique des enregistrements
   - Barre de progression visible
   - Ne fermez pas l'application

3. **Fermeture sécurisée**
   - Libération automatique de l'environnement
   - Fermeture de l'application
   - Retour à votre bureau normal

## 5. Fonctionnalités de Surveillance

### 5.1 Surveillance Vidéo

1. **Reconnaissance faciale continue**
   - Détection de votre présence
   - Vérification de la visibilité du visage
   - Alerte en cas d'absence prolongée

2. **Détection d'objets**
   - Identification des téléphones
   - Détection de tablettes
   - Alerte pour objets suspects

3. **Suivi du regard**
   - Détection des sorties de champ
   - Alerte pour regards prolongés hors écran
   - Analyse des mouvements oculaires

### 5.2 Surveillance Audio

1. **Analyse de la voix**
   - Détection de voix tierces
   - Identification des chuchotements
   - Alerte pour conversations

2. **Détection de bruits**
   - Sons suspects
   - Bruits de clavier anormaux
   - Alerte pour bruits non autorisés

### 5.3 Surveillance d'Écran

1. **Capture d'écran**
   - Enregistrement continu
   - Sauvegarde sécurisée
   - Transmission automatique

2. **Analyse d'activité**
   - Changements d'application
   - Copier-coller fréquent
   - Patterns suspects

## 6. Dépannage

### 6.1 Problèmes Courants

1. **Webcam non détectée**
   ```
   Solution :
   - Vérifiez la connexion USB
   - Autorisez l'accès dans les paramètres
   - Redémarrez l'application
   ```

2. **Microphone non fonctionnel**
   ```
   Solution :
   - Testez dans les paramètres système
   - Vérifiez les permissions
   - Changez de périphérique si nécessaire
   ```

3. **Connexion instable**
   ```
   Solution :
   - Vérifiez votre connexion Internet
   - Fermez les applications gourmandes
   - Contactez le support si persistant
   ```

### 6.2 Erreurs d'Authentification

1. **Reconnaissance faciale échouée**
   - Améliorez l'éclairage
   - Retirez les lunettes de soleil
   - Assurez-vous d'être seul dans le champ

2. **Pièce d'identité non reconnue**
   - Utilisez une pièce officielle
   - Assurez une photo claire
   - Vérifiez la validité du document

### 6.3 Problèmes de Performance

1. **Application lente**
   - Fermez les applications inutiles
   - Vérifiez l'espace disque
   - Redémarrez l'ordinateur

2. **Surveillance interrompue**
   - Vérifiez les permissions système
   - Redémarrez l'application
   - Contactez le support

## 7. FAQ

### 7.1 Questions Générales

**Q : Puis-je utiliser plusieurs écrans ?**
R : Oui, tous vos écrans sont surveillés automatiquement.

**Q : Que se passe-t-il si je perds la connexion ?**
R : L'application continue la surveillance locale et synchronise à la reconnexion.

**Q : Mes données sont-elles sécurisées ?**
R : Oui, toutes les données sont chiffrées et conformes au RGPD.

### 7.2 Questions Techniques

**Q : L'application ralentit-elle mon ordinateur ?**
R : L'impact est minimal, optimisé pour ne pas perturber votre travail.

**Q : Puis-je désactiver temporairement la surveillance ?**
R : Non, la surveillance est continue pendant l'examen pour garantir l'intégrité.

**Q : Que faire si je détecte un bug ?**
R : Contactez immédiatement le support avec une description détaillée.

### 7.3 Questions de Confidentialité

**Q : Qui peut voir mes enregistrements ?**
R : Seuls les instructeurs autorisés et l'administrateur système.

**Q : Combien de temps sont conservées mes données ?**
R : Maximum 90 jours, puis suppression automatique.

**Q : Puis-je demander la suppression de mes données ?**
R : Oui, contactez le support pour exercer vos droits RGPD.
